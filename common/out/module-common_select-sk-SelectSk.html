Class: SelectSk

common/select-sk~SelectSk()

select-sk: a selection element that allows any kind of children. Clicking on the children will cause them to be selected. The select-sk elements monitors for the addition and removal of child elements and will update the 'selected' property as needed. Note that it does not monitor the 'selected' attribute of child elements, and will not update the 'selected' property if they are changed directly.

The detail of the event contains the child element index:

    selection: 1,
  }
