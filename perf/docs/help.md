# Skia Perf help & support

Use this guide to find support channels and report issues with the Performance Dashboard for users outside of Google.

## Reporting Issues & Feature Requests

### File an Issue Report

- **Best for**: Bug reports, feature requests, and regression tracking.
- **Tracker**: File an issue in the [Public Chromium Issue Tracker](https://g-issues.chromium.org/issues/new?component=1547614&template=1970127).
