# anomaly-playground-sk

A page for experimenting with anomaly detection algorithms.

## Features

- Visualize trace data.
- Run different anomaly detection algorithms on the trace data.
- Adjust parameters like radius and threshold.
- See detected anomalies highlighted on the graph.
